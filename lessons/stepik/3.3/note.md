# Плейс холдеры - 

Функция printf берет строку (первый аргумент) и на место очередного %d (format placeholder) подставляет очередной аргумент.
[fplaceholder](image.png)

- Плейсхолдер **%d - целое число**
- Плейсхолдер **%f - дробное число**
