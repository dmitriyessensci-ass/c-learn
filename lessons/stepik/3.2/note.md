
# Каретка

| scape последовательность 	| Значение                                                  |
|---------------------------|-----------------------------------------------------------|
| \n                        | новая строка (курсор на первую позицию следующей строки)  |
| \r 	                    | возврат каретки (курсор на первую позицию этой строки)    |
| \t 	                    | табуляция                                                 |
| \' 	                    | '                                                         |
| \" 	                    | "                                                         |
| \\ 	                    | \                                                         |
